# Scoring System

The scoring module implements the full logic for tracking a padel match, from individual points to the final match result. On this page, we'll dive into the `MatchScore` class and how to progress the state of a match. {{ className: 'lead' }}

## The MatchScore model

The `MatchScore` class is the central entity for managing match progress. It internally tracks points, games, and sets for both teams and handles the complex logic of padel scoring like Deuce and Advantage.

### Enums

Before diving into the class, it is important to know the `TeamId` Enum, which is used to identify the teams in the match.
- `TeamId.A`: Represents the first team.
- `TeamId.B`: Represents the second team.

---

## Add a point {{ tag: 'method', label: 'score.add_point()' }}

<Row>
  <Col>

    This method is the primary way to progress the match. You pass the team that won the point, and the `MatchScore` engine will automatically calculate if that point results in a game, set, or match win.

    Points follow the traditional sequence: `0`, `15`, `30`, `40`.

    If both teams reach 40, the state becomes **Deuce**. Winning a point at Deuce grants **Advantage**. If the team with Advantage loses the next point, the state returns to **Deuce**.

    ### Required arguments

    <Properties>
      <Property name="team" type="TeamId">
        The team that won the point (`TeamId.A` or `TeamId.B`).
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Usage" tag="method" label="add_point">

    ```python
    from padelkit import MatchScore
    from padelkit.scoring import TeamId

    score = MatchScore()
    
    # Team A wins a point
    score.add_point(TeamId.A)
    ```

    </CodeGroup>

  </Col>
</Row>

## Retrieve the current score {{ tag: 'method', label: 'score.get_points()' }}

<Row>
  <Col>

    This method returns the current points for a specific team in the current game.

    ### Required arguments

    <Properties>
      <Property name="team" type="TeamId">
        The team whose points you want to retrieve.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

## Advanced State Management

<Row>
  <Col>

    The `MatchScore` engine automatically handles the progression of games and sets based on the points won.

    - **Games**: Won by reaching 40 and winning the deciding point (or from Advantage).
    - **Sets**: A set is won upon reaching 6 games with a difference of 2, or upon reaching 7. *(Note: Tie-break logic is not implemented in the current initial version)*.
    - **Match**: Configured as best of 3 sets. The first team to win 2 sets wins the match.

  </Col>
  <Col sticky>
    <CodeGroup title="Example" tag="concept" label="progression">
    ```python
    from padelkit import MatchScore
    from padelkit.scoring import TeamId

    score = MatchScore()

    # Simulate Team A winning a flawless game
    score.add_point(TeamId.A) # 15
    score.add_point(TeamId.A) # 30
    score.add_point(TeamId.A) # 40
    score.add_point(TeamId.A) # Game A

    print(score.get_games(TeamId.A))
    # Output: 1
    ```
    </CodeGroup>
  </Col>
</Row>
