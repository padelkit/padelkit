# MatchScoreHistory

`MatchScoreHistory` is the event-sourcing engine that drives the score calculation. It maintains a chronological record of points and recalculates the game state efficiently. {{ className: 'lead' }}

## Properties

<Properties>
  <Property name="teams" type="dict[TeamId, Team] | None">
    The teams assigned to this match history, if any.
  </Property>
  <Property name="starting_server_team" type="TeamId">
    <AddedIn version="2.0.1">
      The team configured to start serving first in the match.
    </AddedIn>
  </Property>
  <Property name="set_starting_servers" type="dict[int, dict[TeamId, Player]]">
    <AddedIn version="2.0.1">
      A dictionary mapping 0-based set indexes to each team's chosen initial server for that set.
    </AddedIn>
  </Property>
</Properties>

## Adding Points {{ tag: 'method', label: 'history.add_point()' }}

<Row>
  <Col>

    Adds a point won by a specific team to the history and returns the new recalculated score.

    ### Required arguments

    <Properties>
      <Property name="team" type="TeamId">
        The team that won the point (`TeamId.A` or `TeamId.B`).
      </Property>
    </Properties>

    ### Returns

    <Properties>
      <Property name="MatchScore" type="MatchScore">
        The new recalculated match state.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Usage" tag="method" label="add_point">

    ```python
    history.add_point(TeamId.A)
    ```

    </CodeGroup>

    ```text {{ title: 'Output structure' }}
    TEAM A |   | 0 | 15
    TEAM B |   | 0 |  0
    ```

  </Col>
</Row>

## Modifying Points {{ tag: 'method', label: 'history.modify_point()' }}

<Row>
  <Col>

    Corrects a mistake by overwriting a point at a specific index in the history, triggering an instant recalculation of the entire match state.

    ### Required arguments

    <Properties>
      <Property name="index" type="int">
        The 0-based index of the point in history to modify.
      </Property>
      <Property name="team" type="TeamId">
        The correct team that won the point.
      </Property>
    </Properties>

    ### Returns

    <Properties>
      <Property name="MatchScore" type="MatchScore">
        The new recalculated match state.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Usage" tag="method" label="modify_point">

    ```python
    # Overwrite the first point to be won by Team B
    history.modify_point(0, TeamId.B)
    ```

    </CodeGroup>

    ```text {{ title: 'Output structure' }}
    TEAM A |   | 0 |  0
    TEAM B |   | 0 | 15  *
    ```

  </Col>
</Row>

## Retrieving Score {{ tag: 'method', label: 'history.get_current_score()' }}

<Row>
  <Col>

    Iterates over the entire point history to compute the score up to the latest point.

    ### Returns

    <Properties>
      <Property name="MatchScore" type="MatchScore">
        The current recalculated match state.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Usage" tag="method" label="get_current_score">


    </CodeGroup>


## Setting Set Starting Servers {{ tag: 'method', label: 'history.set_set_starting_server()' }}

<Row>
  <Col>

    Configures the starting server player for a specific team in a specific set.

    ### Required arguments

    <Properties>
      <Property name="set_index" type="int">
        The 0-based index of the set.
      </Property>
      <Property name="team_id" type="TeamId">
        The team identifying (`TeamId.A` or `TeamId.B`).
      </Property>
      <Property name="player" type="Player">
        The [`Player`](/player) chosen to serve first for the team in that set.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Usage" tag="method" label="set_set_starting_server">

    ```python
    from padelkit import Player, TeamId

    player = Player("Juan Lebrón")
    history.set_set_starting_server(set_index=0, team_id=TeamId.A, player=player)
    ```

    </CodeGroup>

  </Col>
</Row>
