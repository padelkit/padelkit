# Court Geometry

PadelKit defines the official dimensions and landmark positions for a standard padel court based on the International Padel Federation (FIP) standards. {{ className: 'lead' }}

## Dimensions {{ id: 'dimensions' }}

A standard padel court is 20 meters long and 10 meters wide. In PadelKit, you can access these measurements via the `CourtDimensions` class.

```python
from padelkit.court import CourtDimensions

dimensions = CourtDimensions.fip_standard()
print(f"Length: {dimensions.length}m, Width: {dimensions.width}m")
```

## Coordinate System {{ id: 'coordinate-system' }}

PadelKit uses a centralized coordinate system to represent positions on the court:

- **Origin (0,0)**: Located exactly at the center of the court (at the net).
- **X-axis**: Represents the width (from -5.0 to 5.0).
- **Y-axis**: Represents the length (from -10.0 to 10.0).

This system simplifies calculations for player movement and ball positioning.

## Landmarks {{ id: 'landmarks' }}

The `PadelCourt` class provides access to standardized landmarks, such as the corners, service lines, and net positions.

```python
from padelkit import PadelCourt

court = PadelCourt.fip_standard()
net_center = court.point("NET_CENTER")
print(f"Net center coordinates: {net_center}")
```
