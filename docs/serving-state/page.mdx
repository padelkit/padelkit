# ServingState

The `ServingState` dataclass encapsulates the serving team and player at a specific point in time. It is exposed through the `server` property of the [`MatchScore`](/match-score). {{ className: 'lead' }}

## Properties

<Properties>
  <Property name="team" type="TeamId">
    The [`TeamId`](/team) currently serving (`TeamId.A` or `TeamId.B`).
  </Property>
  <Property name="player" type="Player | None">
    The specific [`Player`](/player) currently serving, or `None` if player details are not configured.
  </Property>
</Properties>
